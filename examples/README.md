# Puffy Examples

These are small starter examples for the public engine.

- `langchain_video_rag.py` shows the extract -> wait -> consume flow.
- `n8n_puffy_async_template.json` is a tiny polling workflow skeleton.
- `dify_puffy_video_summary.yml` sketches a Dify-style HTTP chain.
- `open_webui_local_research.md` shows how to use the public engine as a local research ingress.
